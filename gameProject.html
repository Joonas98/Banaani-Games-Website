The Forsaken

This page is for occasional updates of my solo game project called The Forsaken. You can find some devlog / update type of videos in my Youtube.

I am the solo developer of the project. It is made using Unity game engine, C# programming language, A* pathfinding and various other systems / APIs'.

Temporarily you can find most of the source code in my Github. I also occasionally buy systems and APIs made by other people from sales. I am not risking to share this sort of code, even if I have modified it to suit my needs.

I am planning to release the game some day, but it is mostly a hobby project for me. I have learned a lot during this project and I have enjoyed a ton!

This page is last updated 13.10.2023

INFO & PICTURES

The basic concept of the game is fairly generic wave based first-person shooter with some roguelike elements. Currently the enemies are only zombies, but I will be implementing various demons and other monsters in the near future.

What seperates my game from other's will be high amount of different abilities, weapons, upgrades and other features that allow the player to have tons of different playstyles and builds.

The theme, setting and graphic style of my game is not very serious or realistic, but rather unserious. Terrain and models are heavily styled and low-poly. SFX and VFX are somewhat cartoony as well.
