HOMEWIDE ADBLOCK SYSTEM

2023 - Rasperry Pi + Pi-Hole setup

I have 24/7 running Raspberry Pi 3 running some systems in my home. In June of 2023, I setup system called Pi-Hole. This system handles queries sent from my smart devices. If the query domain is included in my blocklists, it get's blocked (query returns 0.0.0.0 IP instead).

I use many blocklists, mostly from Firebog. These blocklists include huge amounts of domains that send ads, malware, tracking or various other unwanted stuff. When the ad can't be loaded, most websites just show nothing or empty space where the ad would be.

Some big sites such as Youtube and Twitch send ads from the same domain as the content itself, but there are other solutions like Chronium based extensions for those.

CHESS WITH AI

2020 - Duo C++ Project

We programmed chess from scratch using C++ with a classmate. Its quite basic chess with all the features you would expect, including castling and en passant. You can also choose for each color if they should be AI or human player, so you can do PvP, PvAI and AIvAI.

The chess AI is very strong. It is made with minimax algorithm and improved further with alpha-beta pruning . At the start, player can choose the search depth for each bot. To simplify, this means how many moves the AI calculates in to the future.

See full source code in Github.

BALLS AND BOXES

2019 - Short solo Project

Balls and Boxes is a simple brick breaker game. It was a short (under 2 weeks) school project that I developed alone.

The ball gets faster with time to make it far more interesting. Later levels have different brick types like some require more hits or some consist of multiple smaller bricks. There are also few basic powerups.

Sound effects and few tricks improve the gameplay, making it surprisingly fun. By few tricks I mean e.g. the player's plane has tuned colliders, so you can control the ball better. Left and right walls are slightly slanted downwards, so the ball doesn't take such a long time to return to player.

GRAINRAIDERS

2019 - First proper game project

My first proper game project. It was made during my first year of school with two other students. Our class voted Grainraiders to be the best out of ten games.

The story and gameplay is very simple. You defend your village's food with a revolver and dynamite bundles. Every five waves, you get a short break to loot. Money is used to refill ammo and dynamites. You lose the game when the bandits are able to steal 10 sacks of food.

The gameplay is quite difficult but fun. After few waves, there will be so many enemies, that it's basically all about timing your dynamites well and finishing survivors with the revolver.

SUMO ROBOT

2018 - IoT / Embedded systems

During first year of school, we practiced some IoT skills. In this specific project, we programmed "Sumo robots" with C to achieve various tasks. The code was programmed into a small circuit board that would be then connected to the robot itself. We also used PuTTY to send data from the robot to PCs.

In this video, we used 5 sensor in the bottom of the robot to identify black and white track, which the robot should follow. Later we also programmed the robots to push each other of a ring shaped arena. This was achieved with the 2 cameras in front of the robot and previously mentioned sensor in the bottom.
